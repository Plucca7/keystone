# Memory index

> One line per memory: a link plus a hook -- the one phrase that tells a future session
> whether the memory is worth opening. This index is what gets loaded; the full files in
> this folder are opened on demand only. The discipline (when to save, update, delete)
> lives in `.claude/rules/long-term-memory.md`.
